## 翻译方法

所有组件都遵循统一的翻译模式：

```typescript
// 1. 导入翻译函数
import { t } from '../../locales';

// 2. 使用翻译
<button>{t.common.save}</button>
<h1>{t.sidebar.video}</h1>
```
