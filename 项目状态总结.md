## 📝 关于"Preferences"面板

截图中显示的"Preferences"面板是**Next.js开发工具**的界面，不是项目代码的一部分。

这个面板包含：
- General（通用设置）
- Theme（主题）
- Position（位置）
- Size（大小）
- Hide Dev Tools（隐藏开发工具）
- Route（路由信息）

**这些是Next.js框架提供的开发者工具，无法通过项目代码翻译。**

### 如何隐藏开发工具

如果你想隐藏这个面板，可以在 `next.config.js` 中添加：

```javascript
module.exports = {
  devIndicators: {
    buildActivity: false,
    buildActivityPosition: 'bottom-right',
  },
}
```

或者直接点击面板中的"Hide Dev Tools for this session"按钮。
