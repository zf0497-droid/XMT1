# 本地媒体功能

此功能允许用户上传并使用自己的媒体文件（视频、图片、音频）到视频编辑器中。

## 功能特性

- 从您的电脑上传视频、图片和音频文件
- 按类型组织媒体（视频、图片、音频）
- 一键将上传的媒体添加到时间轴
- 查看上传文件的缩略图和元数据
- 使用 localStorage 在会话之间保持上传的媒体

## 使用方法

1. 点击侧边栏中的"本地媒体"图标
2. 点击"上传媒体"按钮从您的电脑选择文件
3. 选择一个或多个文件进行上传
4. 使用选项卡按类型浏览您上传的媒体
5. 点击任何媒体项目将其添加到时间轴

## 技术实现

- 媒体文件使用对象 URL 存储在浏览器中
- 为视频和图片生成缩略图
- 尽可能提取媒体元数据（时长、大小）
- 所有上传的媒体都保存在 localStorage 中
- 不再需要时可以单独删除文件

## 支持的文件类型

- 视频：MP4、WebM、Ogg 等
- 图片：JPEG、PNG、GIF、WebP 等
- 音频：MP3、WAV、Ogg 等

## 注意事项

- 媒体存储在浏览器本地，不会上传到任何服务器
- 大文件可能会影响浏览器性能
- 某些文件格式可能不被所有浏览器支持
